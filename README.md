# Cluster-F

Dokumentation für **Cluster-F** – ein ESP32-Umweltsensor, der mit einem BME280 Temperatur, Luftfeuchte und Luftdruck misst und die Werte per WLAN über MQTT an einen Broker sendet.

Dieses Repository enthält die **Doku-Website**. Die vollständige Anleitung (Installation, Konfiguration, Funktionsweise und Fehlerbehebung) liegt in `content/docs/` und wird als Website dargestellt.

## Doku lokal starten

```bash
npm install
npm run dev
```

Anschließend [http://localhost:3000](http://localhost:3000) öffnen.

## Technik

Gebaut mit [Next.js](https://nextjs.org) und [Fumadocs](https://fumadocs.dev) auf Basis des [Unmint](https://github.com/gregce/unmint)-Templates.
