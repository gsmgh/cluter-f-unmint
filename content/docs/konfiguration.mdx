Die wichtigsten Einstellungen befinden sich in der Datei `main.py`.

## MQTT-Broker-IP anpassen

Der MQTT-Broker bezieht seine IP-Adresse per **DHCP** und kann sich daher ändern. Bei einer geänderten IP muss in `main.py` die Konstante `BROKER` angepasst werden:

```python
# main.py
BROKER = "192.168.1.***"   # <-- IP des MQTT-Brokers
```

<Warning title="Broker nicht erreichbar?">
Ändert sich die IP des Brokers, schlägt die Verbindung mit `ECONNABORTED` fehl. In diesem Fall die neue IP ermitteln und oben eintragen – siehe [Fehlerbehebung](/docs/fehlerbehebung).
</Warning>

## MQTT-Topics

Der ESP32 veröffentlicht die Messwerte unter folgenden Topics:

| Topic                | Inhalt           | Einheit |
|----------------------|------------------|---------|
| `sensor/temperatur`  | Temperatur       | °C      |
| `sensor/luftfeuchte` | Relative Feuchte | %       |
| `sensor/luftdruck`   | Luftdruck        | hPa     |

<Tip>
Zum Testen lassen sich die Topics mit einem MQTT-Client wie **MQTT Explorer** oder dem Kommandozeilen-Tool `mosquitto_sub` abonnieren.
</Tip>
